## Project Overview

This project showcases an **interactive Airbnb Tableau Dashboard** built to analyze listings across New York City. The dashboard provides insights into **pricing, room types, neighbourhood trends, reviews, hosts, and seasonal booking patterns** to support data-driven decision-making.
